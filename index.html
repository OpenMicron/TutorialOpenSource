Windows Preparation

1. Install Docker

Navigate to https://www.docker.com on your web browser

Select Download Docker Desktop > Download for Windows - AMD64

Open the downloaded executable and install, make sure to use these options!

Once installed, open the Docker application, you can skip signing in by selecting this option:

2. Clone Docker container

On the Docker window, go to the search tab and write "hpretl/iic-osic-tools".

Select "Pull" and wait for Docker to download the container.

Be patient! This process can take minutes depending on your computer and network speed.

You will know the cloning is done when a green checkmark appears

3. Enable WSL

Open the Windows terminal

Copy the following command and press the Enter key

wsl --install

When finished executing, restart your computer. After logging back in, open the terminal again and execute:

wsl --install Ubuntu

This command will install WSL Ubuntu on your computer; it will prompt for a user and password. Remember them! These will be your credentials to access the tools.

Once done, open Docker again and enable WSL integration. Do not forget to click Apply.

4. Setup Ubuntu environment

Open the Ubuntu terminal by typing "Ubuntu" in the search tab

Type the following commands in the Ubuntu terminal, type your password and type "Y" if required

sudo chmod 666 /var/run/docker.sock
sudo apt install socat

After that, run the following commands

cd
git clone https://github.com/iic-jku/iic-osic-tools

macOS Preparation

1. Install Docker

Navigate to https://www.docker.com on your web browser

Select Download Docker Desktop > Download for Mac

Open the downloaded .dmg file and drag docker to the Applications folder

2. Clone docker container

Open Docker via the Applications menu or by pressing Cmd + Spacebar and typing docker.

You can skip signing in by selecting this option

Go to the search tab and write "hpretl/iic-osic-tools". Select "Pull" and wait for Docker to download the container.

Be patient! This process can take minutes depending on your computer and network speed.

You will know the cloning is done when a green checkmark appears

3. Download XQuartz

Navigate to https://www.xquartz.org on your browser and select XQuartz-2.8.5.pkg, install it the same way as you installed Docker.

Open XQuartz and paste the following command in the window

xhost + 127.0.0.1

4. Clone github repository

Open a Mac terminal by pressing Cmd + Spacebar and typing "terminal", paste and run the following commands one by one. Type your password if required:

cd
git clone https://github.com/iic-jku/iic-osic-tools.git
sudo chmod 666 /var/run/docker.sock

Linux Preparation

These instructions are for users with a Debian-based distro (Ubuntu, Mint, Pop, Kali, Debian). Users with a different distribution (RedHat, Fedora, Arch) will have to follow the same steps with their own package manager.

1. Install Docker

Open this webpage in your browser Docker Desktop release page

Scroll down and click the DEB package button to download the latest .deb file

Open your terminal and navigate to the folder where the .deb file was downloaded.

Run the apt-get install command on the .deb file:

sudo apt-get update
sudo apt-get install ./docker-desktop-amd64.deb

At the end of the installation process, apt displays an error (N: Download is performed unsandboxed as root...) due to installing a downloaded package. You can ignore this error message

2. Launch Docker

Navigate to the Docker Desktop application in your Gnome/KDE Desktop and open it (type your password if required).

Go to the search tab and write "hpretl/iic-osic-tools". Select "Pull" and wait for Docker to download the container.

Be patient! This process can take minutes depending on your computer and network speed.

You will know the cloning is done when a green checkmark appears

3. Clone github repository

Open a terminal and type the following commands (type your password if required):

cd
git clone https://github.com/iic-jku/iic-osic-tools.git
sudo chmod 666 /var/run/docker.sock

Step 2: Launch the Design Tools

Congratulations! You're all set for designing. Once you have finished the preparation process, launching the tools is very straightforward.

Make sure Docker is running and open the Ubuntu terminal by typing "Ubuntu" in the search tab.

Make sure Docker and XQuartz are running and open the Terminal by pressing Cmd + Spacebar and typing "Terminal".

Make sure Docker is running and open a Terminal.

Type the following command in the terminal:

~/IIC-OSIC-TOOLS/start_x.sh
~/iic-osic-tools/start_x.sh
~/IIC-OSIC-TOOLS/start_x.sh

If the following message appears, just press the letter "s":

And you're done! A new terminal will open; this will be your main contact with the design tools.

In this new terminal, type the following command to select the Skywater130 PDK (You will have to do this each time you launch a new terminal window or tab!):

sak-pdk sky130A

To open the schematics tool, type the following command:

xschem

To open the layout tool, type the following command:

klayout -e

Step 3: Verification and course database

In this last step you will download the course content database, run a simulation and open a stdcell layout.

And open the schematics tool by typing the following commands in the design terminal:
