Windows Preparation

1. Install Docker

Navigate to https://www.docker.com on your web browser

Select Download Docker Desktop > Download for Windows - AMD64

Open the downloaded executable and install, make sure to use these options!

Once installed, open the Docker application, you can skip signing in by selecting this option:

2. Clone Docker container

On the Docker window, go to the search tab and write "hpretl/iic-osic-tools".

Select "Pull" and wait for Docker to download the container.

Be patient! This process can take minutes depending on your computer and network speed.

You will know the cloning is done when a green checkmark appears

2. Enable WSL

Open the Windows terminal

Copy the following command and press the Enter key

wsl --install

When finished executing, restart your computer; After logging back in, open the terminal again and execute:

wsl --install Ubuntu

This command will install WSL Ubuntu in your computer, it will prompt for a user and password. Remember them! These will be your credentials to access the tools.

Once done, open Docker again and enable WSL integration. Do not forget to click Apply.

3. Setup Ubuntu environment

Open the Ubuntu terminal by typing "Ubuntu" in the search tab

Type the following commands in the Ubuntu terminal, type your password and type "Y" if required

sudo chmod 666 /var/run/docker.sock
sudo apt install socat
After that, run the following commands 
cd
git clone https://github.com/iic-jku/iic-osic-tools

Congratulations! You're all set for designing.

How to launch the tools

Once you finished the preparation process, the process to launch the tools is very straightforward

Make sure Docker is running and open the Ubuntu terminal by typing "Ubuntu" in the search tab

Type the following command in the terminal:

~/IIC-OSIC-TOOLS/start_x.sh

If the following message appears, just press the letter "s"

And you're done! A new terminal will open, this will be your main contact with the design tools

In this new terminal, type the following command to select the Skywater130 PDK

sak-pdk sky130A

To open the schematics tool, type the following command:

xschem

To open the layout tool, type the following command:

klayout -e

macOS Preparation

1. Install Docker

Navigate to https://www.docker.com on your web browser

Select Download Docker Desktop > Download for Mac

2. Clone docker container

Open Docker, go to the search tab and write "hpretl/iic-osic-tools".

Select "Pull" and wait for Docker to download the container.

3. Clone github repository

Open a Mac terminal, paste and run the following commands:

mkdir ~/open-source
cd ~/open-source
git clone https://github.com/iic-jku/iic-osic-tools.git
cd iic-osic-tools
sudo chmod 666 /var/run/docker.sock

3. Download XQuartz

Navigate to https://www.xquartz.org and select XQuartz-2.8.5.pkg

Open XQuartz and paste the following command in the window

xhost + 127.0.0.1
